# counting 

A personal life operating system focusing on emotional awareness and zero-friction tracking. 

## 🌟 核心理念 (Core Philosophy)
传统的时间管理工具往往因过度强调“打卡”而引发内耗。**counting** 摒弃了机械的流水账，将记录的粒度重塑为“情绪觉察”与“能量管理”。它接纳长达 2 小时的枯坐，也赋予 0 秒“瞬间闪念”同等的生命重量。

## ✨ 核心产品力 (Key Features)
*   **零阻力快门 (Zero-Friction Capture)**：突破层级嵌套，融合系统原生手势与麦克风语音转文字（SFSpeechRecognizer），将灵感定格路径压缩至极简。
*   **动态能量光谱 (Energy Aura Bar)**：UI 主题色不再是冷冰冰的静态设置，而是根据用户当前主导的情绪状态实时重绘，呈现极具生命力的气场。
*   **生命热力图谱 (365-Day Life Heatmap)**：基于 Apple 原生 Calendar API 打造时间线日视图与年视图，直观映射用户的心理周期与能量流转。
*   **掘金归档 (Archive & Reflection)**：重构传统的“删除”逻辑。允许将过往足迹沉淀为高价值的生命坐标，完成 Local-First（本地优先）的闭环心理建设。

## 🛠 技术栈 (Tech Stack)
*   **Framework**: SwiftUI / MVVM Architecture
*   **Native Capabilities**: AVFoundation, PhotosUI, SFSpeechRecognizer
*   **Data Persistence**: JSON-based Local Storage (Privacy First)
